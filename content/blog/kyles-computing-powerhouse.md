## The Build

This is the machine behind everything I build. Every multi-agent system, every LLM pipeline, every project on this site — it all runs on this rig. I designed it from the ground up for AI workloads: local model inference, large dataset processing, and running 20+ agent orchestration systems simultaneously.

Here's the full spec sheet.

## Core System

| Component | Spec |
|-----------|------|
| **CPU** | Intel Core i9-14900F — 2.0 GHz, 24-Core |
| **CPU Cooler** | NZXT Kraken 360 RGB — 84.13 CFM Liquid Cooling |
| **Thermal Paste** | Arctic Silver 5 High-Density Polysynthetic Silver (3.5g) |
| **Motherboard** | Gigabyte Z790 GAMING X AX — ATX, LGA1700 |
| **Memory** | Corsair Vengeance 64 GB (2 x 32 GB) DDR5-5600 CL36 |

## Graphics

| Component | Spec |
|-----------|------|
| **GPU** | Gigabyte WINDFORCE GeForce RTX 5090 — 32 GB VRAM |

The RTX 5090 is the centerpiece. 32 GB of VRAM means I can run large local models without quantization compromises, and it handles the inference workloads for projects like OpenClaw and Phantom Depth without breaking a sweat.

## Storage — 30+ TB Total

| Drive | Capacity | Type |
|-------|----------|------|
| WD Black SN850X | 2 TB | M.2 NVMe PCIe 4.0 (Primary OS) |
| Samsung 990 PRO | 2 TB | M.2 NVMe PCIe 4.0 |
| Samsung 990 PRO | 4 TB | M.2 NVMe PCIe 4.0 |
| Samsung 870 EVO | 1 TB | 2.5" SATA SSD |
| Seagate Barracuda | 4 TB | 3.5" HDD (5400 RPM) |
| Seagate Barracuda | 8 TB | 3.5" HDD (5400 RPM) |
| Seagate Barracuda | 10 TB | 3.5" HDD (5400 RPM) |

NVMe drives handle active projects, model weights, and datasets. The HDDs serve as bulk storage and local backup. When you're working with large language models and multi-terabyte datasets, you burn through storage fast — so I built for capacity from day one.

## Chassis & Power

| Component | Spec |
|-----------|------|
| **Case** | Corsair 7000D Airflow — ATX Full Tower |
| **PSU** | Corsair RM1000e (2023) — 1000W, 80+ Gold, Fully Modular |
| **OS** | Microsoft Windows 11 Pro (OEM, 64-bit) |

The 7000D is massive, but that's the point. Full tower gives room for all the drives, proper cable management, and unobstructed airflow. The 1000W PSU handles the RTX 5090's power draw with headroom to spare.

## Cooling

| Component | Qty |
|-----------|-----|
| Noctua A14 PWM chromax.black.swap — 140mm | x4 |
| Noctua NF-A12x25 PWM chromax.black.swap — 120mm | x3 |

Seven Noctua fans total, plus the 360mm AIO on the CPU. Noctuas are the gold standard for a reason — near-silent even under full load. The system stays cool during sustained GPU inference runs that would cook lesser builds.

A QD-OLED panel at 4K and 240Hz. Perfect pixel contrast for dark-themed code editors and terminal work, and the 240Hz refresh rate makes everything feel instant.

## Why This Build

I didn't build a gaming PC that happens to do AI work. I built an AI workstation that happens to be great for everything else. Every component choice was driven by the same question: *will this let me run more models, process more data, and iterate faster?*

The answer is yes.
