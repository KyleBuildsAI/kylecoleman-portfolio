## Overview

OpenSprite is a web-based tool for importing, previewing, and testing sprite assets before integrating them into a project. Load your own sprite sheets or explore the built-in examples. Randomize animations, adjust parameters, and visually validate your assets — without spinning up a full dev environment.

![OpenSprite — Full Interface](/opensprite-main.png)

## Origin Story

Originally built to support the development of AgentHQ, the 2D visualization layer for the OpenClaw multi-agent system. After purchasing an asset pack, I needed a fast way to inspect and iterate on sprites before committing them to the project. OpenSprite solved that problem and became a standalone tool in the process.

![OpenSprite — Drop Zone](/blank-opensprite.png)

## Built for the Ecosystem

Pairs naturally with OpenClaw for anyone building 2D agent visualizations. Fully open source — fork it, extend it, and add your own features.
