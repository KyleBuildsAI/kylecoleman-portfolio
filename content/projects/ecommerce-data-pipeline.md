## Overview

Custom-built system that collects and analyzes eCommerce data hourly instead of weekly. Reduced manual task time by 50% and enabled data-driven decisions that drove 15% revenue growth.

## Keyword-to-Product Matching Pipeline

The core challenge: a manual keyword review process that required cycling through 100,000+ keywords on a 65-day refresh cadence. The system I built automates the entire workflow — it ingests the full keyword set, matches each keyword against relevant products, and scores whether the pairing is a strong fit.

## AI-Powered Product Analysis

To power accurate matching, I integrated AI to analyze product listings and generate structured product cards. The system extracts key attributes like dimensions, product category, and other relevant metadata. These cards feed into a relevance scoring API that evaluates how closely each keyword aligns with the matched product, automatically filtering out low-relevance pairings.

## The Result

A process that eliminated hours of repetitive, manual keyword review and replaced it with a scalable, repeatable workflow. What used to take days of manual cycling through spreadsheets now runs automatically on a schedule — freeing up time for strategy instead of data entry.
