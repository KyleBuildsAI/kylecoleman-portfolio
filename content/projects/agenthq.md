## Overview

AgentHQ is an isometric pixel art visualization that brings AI agent activity to life. Watch agents work in real time across specialized zones — the Server Core, Collab Hub, Library, Workstation, and Orchestra — each representing a different function in the multi-agent system.

![AgentHQ — Early Demo](/AgentHQ-Early-Demo.png)

The early prototype above shows the initial layout before the full visual overhaul. The final version features detailed pixel art environments, named agents with role-based coloring, and a live activity feed showing exactly what each agent is doing in real time.

## What It Demonstrates

AgentHQ takes the abstract concept of multi-agent orchestration and makes it tangible. Instead of reading logs or monitoring dashboards, you can *see* your agents — the Coder building REST API endpoints, the Analyst running prediction models, the Publisher drafting content, the Orchestrator coordinating sprints — all working simultaneously in a space that feels alive.
